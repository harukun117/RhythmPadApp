# RhythmPadアプリ（まだリリースしていません)
## 概要
録音した音声を、ボタンで再生できるサンプラーアプリです。
SwiftUIの勉強の一環として作成したアプリケーションです。
## 開発言語や技術
・Swift

・SwiftUI

・Realm

## 機能
・音声の録音機能

・音声の切り取り

・最大12個のボタンで音声を再生することができる

・同時再生可能

## こんな人におすすめ
・幼児に遊ばせたい人

・録音した音声や音を使って音楽を作りたい人

・簡易的なシンセサイザーとして使いたい人
